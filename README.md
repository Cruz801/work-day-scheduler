## work-day-scheduler

# Purpose
To plan your day accordingly and help you stay on track to accomplish all that you seek in the Day!
Future blocks will show blue,current time blocks show red, and past time blocks show gray.

# Built With
* HTML
* CSS
* Jquery
* Bootstrap
